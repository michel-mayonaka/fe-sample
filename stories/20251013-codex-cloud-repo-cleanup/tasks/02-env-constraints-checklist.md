# 02 環境制約と適合チェックリスト

## 目的
codex-cloud の実行制約（FS/ネットワーク/承認）を明文化し、それに適合する運用チェックリストを用意する。

## 制約（想定）
- Filesystem: workspace 書込のみ（外部パス不可）。
- Network: 制限あり（依存取得は不可前提）。
- Approvals: 一部コマンドは承認要求となる。

## チェックリスト（ドラフト）
- [ ] `MCP_OFFLINE=1` で依存解決不要（vendor 化）。
- [ ] 生成物は `out/` または `site/` に集約。
- [ ] ネットワーク required な手順は代替（スキップ/分離）を明示。
- [ ] スモークチェックは 2 分以内に完了。
