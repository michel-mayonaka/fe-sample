# 005 テスト整備（マッピング/モック）

## 目的
- 入力マッピングと状態遷移のユニットテストを整備し、回帰を防止する。

## スコープ
- `pkg/game/input` のテスト：状態合成/デッドゾーン/エッジ検出。
- モック `InputSource` のテストユーティリティ。

## 成果物
- `pkg/game/input/..._test.go` とモック実装 `testing` 用ヘルパ。

## 受け入れ基準
- [x] `pkg/game/input` のユニットテストが追加され、押下/長押し/同時押しのケースを網羅。
- [x] マッピング（矢印/決定/キャンセル/1..3）が `DefaultLayout` に含まれることを検証。
